# UPDATE

> For **WHERE** examples, please see the **WHERE** section.

## Simple update

### QueryBuilder

```ts
// highlight-start
import { QueryBuilder } from 'mysql2-orm';
// highlight-end

const { sql, params } = QueryBuilder.update({
  table: 'users',
  set: {
    name: 'John',
  },
});
```

### MySQL2 ORM Class

```ts
// highlight-start
import { MySQL } from 'mysql2-orm';

const pool = new MySQL({
  // ...
});
// highlight-end

await pool.update({
  table: 'users',
  set: {
    name: 'John',
  },
});
```

> SQL Query

```sql
UPDATE `users` SET `name` = ?
-- params: ['John']
```
